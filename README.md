Interactive Housing Data Analysis & Machine Learning Platform
Overview

This project is an interactive web application built with Streamlit that allows users to:

Upload housing datasets
Clean and preprocess data
Handle missing values
Perform Exploratory Data Analysis (EDA)
Visualize data
Train Machine Learning models
Evaluate model performance

The application provides an end-to-end data science workflow without writing code.

Project Goal

Real estate datasets often contain:

Missing values
Inconsistent formats
Mixed data types

This application enables users to transform raw housing data into a machine-learning-ready dataset through an interactive interface.

Main Features
Dataset Exploration

Users can:

Preview records
View dataset dimensions
Display column names
Generate descriptive statistics
Data Cleaning

Supports:

Missing value treatment
Duplicate removal
Column removal
Row filtering
Data type conversion
Data Transformation

Includes:

One-Hot Encoding

For nominal categorical variables.

Label Encoding

For ordinal categorical variables.

Automatic Numeric Extraction

Converts values such as:

₹ 1,200,000
1200 sqft

into usable numeric values.

Interactive Data Editing

Users can:

Edit records
Add new rows
Update values

directly inside the application.

Data Visualization

The application generates:

Histogram

Distribution analysis of numerical features.

Scatter Plot

Relationship between:

Property Size
Property Price
Box Plot

Outlier detection.

Correlation Heatmap

Feature relationship analysis.

Machine Learning Module

Users can train models directly from the interface.

Supported models:

Random Forest Classifier
Support Vector Machine (SVM)
XGBoost Classifier

(if installed)

Model Evaluation

The application automatically displays:

Accuracy Score
Classification Report
Performance Metrics
